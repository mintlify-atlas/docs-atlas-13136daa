Experimental is a BepInEx plugin for Gorilla Tag, authored by Astra, that consolidates eight distinct movement and utility mods into a single, lightweight package. Instead of juggling multiple DLL files and separate menus, you get one plugin (`Astras.Mods.Experimental`) with a unified in-game window that lets you toggle and configure every included mod without ever leaving your session.

## How the Main Menu Works

Once the plugin is loaded, press **E** on your keyboard at any time in-game to open or close the Experimental main window. The window is draggable, so you can reposition it anywhere on screen without blocking gameplay.

Inside the main window there are two tabs:

- **GUIs** — contains an on/off toggle for each of the eight bundled mod windows. Enabling a toggle opens that mod's own draggable configuration panel where you can adjust sliders, presets, and input bindings.
- **Utils** — provides room join and leave utilities powered by Photon, letting you connect to a specific room code directly without using the in-game lobby board.

<Note>
  Pressing **E** again while the main window is open will hide it. Any mod windows you have already opened will remain visible and active — closing the main menu does not disable your mods.
</Note>
