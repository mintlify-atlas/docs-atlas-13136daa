This guide picks up right where [Installation](installation) left off. With BepInEx installed and `Experimental.dll` sitting in your `plugins/` folder, you are ready to open the menu and start experimenting. The whole flow from launch to an active mod takes under two minutes.

## Steps

<Steps>
  <Step title="Launch Gorilla Tag and wait for the lobby">
    Start the game as normal and let it finish loading until you are standing in the main lobby. The plugin loads silently in the background — you will not see a notification, but Experimental is already active.
  </Step>

  <Step title="Press E to open the Experimental window">
    Press **E** on your keyboard. A draggable window labelled **Experimental** appears on screen. You can click and drag the title bar to reposition it anywhere that does not block your view.
  </Step>

  <Step title="Click the GUIs tab">
    The main window has two tabs at the top: **GUIs** and **Utils**. Click **GUIs** to see the full list of mod toggles:

    | Toggle | Mod |
    |---|---|
    | Astras PM V1 | Pull Mod |
    | Apreds UI | Hand Prediction |
    | Gorilla TimeV2 | Time Control |
    | Astras PSA Mod | Joystick Surface Acceleration |
    | Astras VelMax Mod | Surface Velocity Multiplier |
    | AWallWalk V2 | Wall Walking |
    | Astras TurnMod | Smooth / Snap Turning |
    | Astras PitGeo GUI | Pit Geometry Tweaks |

    Every toggle is off by default. Enabling one opens that mod's own configuration window.
  </Step>

  <Step title="Toggle Astras PM V1 to open the Pull Mod window">
    Click the **Astras PM V1** toggle. A new draggable window — the Pull Mod panel — appears on screen. This is where you configure the velocity pull that fires when you release a grabbed surface.
  </Step>

  <Step title="Configure Pull Mod: sliders, preset, and input">
    Inside the Pull Mod window:

    1. **Drag the PullSpeed slider** to set how hard the pull launches you on release. Higher values mean a larger velocity boost.
    2. **Drag the UphillPower slider** to control the upward component of the pull separately.
    3. **Choose a preset** for a quick starting point — for example, click **Legit Setting: (70)** to use a community-balanced value that keeps movement feeling natural.
    4. **Pick an input binding** using the dropdown under **Change Input:**. The available options include Right Grab, Left Grab, Right Trigger, Left Trigger, and joystick or face-button combinations. The selected input must be held for the pull to trigger on hand release.

    <Tip>
      The **Reset** button inside any mod window always returns that mod's settings to their safe defaults. If something feels off, hit Reset before adjusting further.
    </Tip>
  </Step>

  <Step title="Enable a second mod: Astras VelMax Mod">
    Switch back to the Experimental main window, stay on the **GUIs** tab, and click the **Astras VelMax Mod** toggle. The VelMax configuration window opens alongside the Pull Mod window. Both mods are now active simultaneously — you can run as many at once as you like.
  </Step>

  <Step title="Hide the main window by pressing E again">
    Press **E** once more. The Experimental main window closes, giving you an unobstructed view. Your mod windows (Pull Mod, VelMax, and any others you opened) stay visible and remain fully active — hiding the menu does not pause or disable anything.
  </Step>
</Steps>

## Tips

<Tip>
  Every mod window is independently draggable. Arrange them around the edges of your screen so they are accessible without covering the centre of your view.
</Tip>

<Tip>
  Presets are the fastest way to find a good starting value. Pick one close to what you want, then fine-tune with the slider. The **Reset** button is always available to undo any changes and return to safe defaults.
</Tip>

<Tip>
  The **Utils** tab in the main window lets you join a Gorilla Tag room by typing its code directly and clicking Join — no need to walk up to the lobby board or use the phone. You can also leave your current room from the same tab.
</Tip>
